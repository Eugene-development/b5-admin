# Руководство по очистке кеша в b5-admin

## Проблема

После обновления компонентов на современные модальные окна, в браузере может отображаться старая версия из-за кеширования.

## Решение

### 1. Жесткая перезагрузка страницы

**Windows/Linux:**
- `Ctrl + Shift + R` или `Ctrl + F5`

**macOS:**
- `Cmd + Shift + R`

### 2. Очистка кеша браузера

#### Chrome/Edge
1. Открыть DevTools: `F12` или `Cmd/Ctrl + Shift + I`
2. Правый клик на кнопке обновления
3. Выбрать "Очистить кеш и жесткая перезагрузка"

#### Firefox
1. Открыть DevTools: `F12`
2. Правый клик на кнопке обновления
3. Выбрать "Очистить кеш"

### 3. Очистка кеша SvelteKit

Если проблема сохраняется, очистите кеш сборки:

```bash
cd b5-admin
rm -rf .svelte-kit
rm -rf build
npm run dev
```

### 4. Очистка node_modules (крайний случай)

```bash
cd b5-admin
rm -rf node_modules
rm package-lock.json
npm install
npm run dev
```

## Проверка обновлений

После очистки кеша проверьте следующие страницы:

### Контрагенты
- `/delivery` - Службы доставки
- `/contractors` - Подрядчики
- `/services` - Сервисы
- `/suppliers` - Поставщики

Все должны открывать **модальное окно** при нажатии кнопки "Добавить".

### Управление
- `/agents` - Агенты
- `/clients` - Клиенты (только редактирование)

### Бизнес-процессы
- `/contracts` - Контракты

## Признаки успешного обновления

✅ При нажатии "Добавить" открывается **темное модальное окно** по центру экрана  
✅ Форма имеет современный дизайн с валидацией  
✅ Кнопки "Добавить компанию" и "Отмена" внизу формы  
✅ Можно закрыть модальное окно по ESC или клику вне окна  

## Признаки старой версии (кеш не очищен)

❌ Форма отображается inline на странице  
❌ Старый дизайн без темного фона  
❌ Форма не закрывается по ESC
