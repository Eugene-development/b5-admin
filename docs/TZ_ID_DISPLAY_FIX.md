# Исправление отображения ID техзадания

## Проблема

В нескольких местах интерфейса техзаданий отображался внутренний ID техзадания вместо номера проекта, что было неинформативно для пользователей.

## Решение

Заменено отображение ID техзадания на номер проекта в следующих местах:

### 1. Модальное окно редактирования (TzEditModal.svelte)

**Было:**
```svelte
<p class="mt-0.5 text-sm text-violet-100">ТЗ #{tz?.id}</p>
```

**Стало:**
```svelte
<p class="mt-0.5 text-sm text-violet-100">
    {tz?.project?.project_number || `Проект #${tz?.project_id}`}
</p>
```

### 2. Модальное окно удаления (+page.svelte)

**Было:**
```javascript
message: `Вы уверены, что хотите НАВСЕГДА удалить техзадание #${tz.id}?...`
```

**Стало:**
```javascript
const projectInfo = tz.project?.project_number || `Проект #${tz.project_id}`;
message: `Вы уверены, что хотите НАВСЕГДА удалить техзадание для проекта ${projectInfo}?...`
```

### 3. Сообщение об успешном удалении (+page.svelte)

**Было:**
```javascript
addSuccessToast(`Техзадание #${tz.id} удалено`);
```

**Стало:**
```javascript
const projectInfo = tz.project?.project_number || `Проект #${tz.project_id}`;
addSuccessToast(`Техзадание для ${projectInfo} удалено`);
```

## Логика отображения

Во всех случаях используется единая логика:
1. Если у проекта есть `project_number` - отображается он
2. Если номера нет - отображается `Проект #ID_проекта`

Это делает интерфейс более понятным и информативным для пользователей.

## Файлы

- `b5-admin/src/lib/components/business-processes/tz/TzEditModal.svelte`
- `b5-admin/src/routes/(protected)/(business-processes)/tz/+page.svelte`

## Тестирование

1. **Модальное окно редактирования:**
   - Открыть страницу Техзадания
   - Нажать "Редактировать" для любого техзадания
   - Проверить, что в подзаголовке отображается номер проекта

2. **Модальное окно удаления:**
   - Открыть страницу Техзадания
   - Нажать "Удалить" для любого техзадания
   - Проверить, что в сообщении упоминается номер проекта, а не ID техзадания

3. **Сообщение об успешном удалении:**
   - Удалить техзадание
   - Проверить, что в toast-сообщении отображается номер проекта
