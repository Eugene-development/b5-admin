## Выполненные изменения

Изменён фон строки поиска с белого на чёрный:

## Преимущества

1. **Единообразие** - все строки поиска теперь используют один компонент
2. **Упрощение поддержки** - изменения в одном месте применяются везде
3. **Уменьшение дублирования кода** - удалено ~500+ строк дублирующегося HTML
4. **Улучшенная доступность** - компонент включает ARIA-атрибуты и поддержку клавиатуры
5. **Единый стиль** - чёрный фон на всех страницах

## Использование компонента

```svelte
<script>
  import { SearchBar } from '$lib';
  
  let searchTerm = $state('');
  
  function handleSearch(term) {
    // Логика поиска
  }
</script>

<SearchBar bind:value={searchTerm} onSearch={handleSearch} />
```

### Параметры

- `value` - текущее значение поиска (двустороннее связывание)
- `onSearch` - callback-функция, вызываемая при изменении (с debounce 300ms)
- `placeholder` - текст placeholder (по умолчанию: "Поиск по таблице...")

## Особенности компонента

- Автоматический debounce (300ms)
- Кнопка очистки поиска
- Поддержка клавиши Escape для очистки
- ARIA-атрибуты для доступности
- Анонсы для screen readers
- Адаптивный дизайн
